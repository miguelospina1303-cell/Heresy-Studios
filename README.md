# Romero Estudio Premium - Sistema Web Completo

Sistema web profesional para barbería, salón de belleza, tienda de productos y academia de formación.

## 🚀 Características Principales

### Para Clientes:
- Landing Page Premium - Diseño elegante negro/dorado con animaciones
- Sistema de Reservas - Booking por barbero con horarios dinámicos
- Cross-selling Inteligente - Recomendaciones durante el proceso de reserva
- Sistema de Puntos - Programa de fidelización automático
- Tienda Online - Catálogo de productos con carrito de compras
- Academia - Showcase de cursos con inscripción a Hotmart

### Para Administradores:
- Dashboard Completo - Estadísticas en tiempo real con gráficos
- Gestión de Barberos - Rendimiento individual y métricas
- Control de Reservas - Tabla completa con filtros y exportación
- Base de Clientes - Historial, puntos y visitas
- Exportación Excel - Datos completos descargables
- Autenticación Segura - Sistema de login protegido

## 🛠️ Tecnologías Utilizadas

- HTML5 - Estructura semántica
- CSS3 - Estilos avanzados con variables CSS
- JavaScript Vanilla - Lógica sin frameworks
- Chart.js - Gráficos y estadísticas
- SheetJS - Exportación a Excel
- Font Awesome - Iconografía
- Google Fonts - Tipografías (Playfair Display + Inter)
- localStorage - Persistencia de datos local

## 🚀 Cómo Usar

### 1. Instalación Local
Descomprime todos los archivos en una carpeta
Abre el archivo index.html en tu navegador

### 2. Acceso Admin
- URL: admin.html
- Usuario: admin
- Contraseña: romero2024

### 3. Flujo de Reserva
1. Cliente selecciona barbero
2. Elige servicio con cross-selling
3. Selecciona fecha y horarios disponibles
4. Completa datos personales
5. Sistema calcula puntos automáticamente
6. Confirmación de reserva
